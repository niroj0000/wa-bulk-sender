How it works

Four steps to bulk sending

No technical knowledge needed. If you can use Excel, you can use this.

📋
01

Add your contacts

Put phone numbers and names in a simple CSV file. Like an Excel sheet.

✏️
02

Write your message

Edit message.txt with your template. Use {name} and {amount} for personalization.

📱
03

Scan QR once

Run the bot, scan the QR code with your WhatsApp. Takes 5 seconds.

🚀
04

Messages send automatically

Sit back. The bot sends to everyone with a safe delay between each message.

Features

Everything you need

Built for real driver outreach operations. Simple and reliable.

🎯

Personalized messages

Each driver gets their own name, payment amount, and custom details automatically filled in.

🛡️

Ban protection

Built-in 10 second delay between messages to keep your WhatsApp account safe.

📊

Sent/Failed tracking

See exactly which numbers succeeded and which failed. Full log in your terminal.

🔁

Session memory

Scan QR once — the bot remembers your login for next time. No re-scanning needed.

📂

CSV contacts

Edit your contact list in Excel or Notepad. No database, no complex setup.

💸

Completely free

No monthly fees, no WhatsApp Business API costs. Runs on your own laptop.

Installation

Up and running in 5 minutes

Requires Node.js and Google Chrome installed on your computer.

Windows
#1. Clone the repository
$git clone https://github.com/niroj0000/wa-bulk-sender.git
#2. Go into the folder
$cd wa-bulk-sender
#3. Install dependencies
$npm install
#4. Add your contacts to contacts.csv, edit message.txt
#5. Run the bot
$node sender.js
